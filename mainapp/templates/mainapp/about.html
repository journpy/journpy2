About Us

Our mission is to help fast track your tech journey with Python.

JournPy is a registered start up that aims to revolutionise the way persons learn coding. Through an integrated learning approach, we create links between various disciplines and coding thus empowering learners with dynamic skills that are much needed in today's world.

At JournPy we believe that learners should be able to actively interact with their tutors but we equally believe that the future of learning is 100% online. This is why we offer live virtual classes in real time instead of pre-recorded lessons.

JournPy is developed and maintained by Uchenna Adubasim - a Software Engineer working primarily with Python but also uses other languages from time to time. Uchenna is an active contributor on Stack Oveflow - the number one Q/A site for developers looking for help to technical problems.

If you want personalised lessons, you can express your interest using the Contact Form.
